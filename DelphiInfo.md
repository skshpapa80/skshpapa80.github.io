### Delphi 란?

Delphi(델파이)는 Object Pascal(오브젝트 파스칼)을 사용하는 프로그래밍 도구 입니다. 

윈도우에서 주로 사용되는 개발툴이었고 컴포넌트 중심으로 개발되어 UI 기능과 개발속도가 빠른 개발툴입니다. 

그리고 델파이를 만든 헤일스 버그는 마이크로 소프트에서 C#을 개발했기 때문에  C#을 배우기전에 배워두면 나쁘지 않은 언어 입니다. 

하지만 2000년대 초반에 많이 사용한 언어이고 지금은 신규 사용자가 많이 줄었습니다. 

오래된 만큼 많은곳에서 사용되었고 아직 도 많은곳에서 사용하지만 신규 사용자가 없어서 

최근 신규로 생기는 프로젝트에서는 많이 사용하지 않는거 같습니다. 

* 델파이의 오픈소스 버전인 라자루스(Lazanus)가 있고 많은 발전되어서 쓸만한 개발툴이 되어 갑니다.
