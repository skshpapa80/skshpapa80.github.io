#### 기초 설정
제가 가진 태블릿 태클라스트 X80 power S 는 원래 듀얼 OS 태블릿 입니다.

최근에 remex OS 설치후(안드로이드 전용으로) 잘사용하고 있었는데 이번에 만드는 프로그램이 윈도우 태블릿 에서 돌아가는 프로그램을 만들고 있어서 테스트용으로 윈도우10전용으로 OS를 재설치했습니다.

재설치하고 나서 기초 설정 한부분은 기록으로 남김니다.

#### 설정부분

windows 관리도구 -> 디스크 정리에서 C 드라이브 정리해주기 (용량확보)

설정 -> 개인정보 -> 백그라운드 앱 – 모드 끄기 설정 – 백그라운드 앱에서 굳이 사용하지 않는 앱은 꺼줍니다.

윈도우 디펜터 자동 샘플전송 옵션 끄기 – 실시간 보호화 클라우드 기반 보호는 필요한꺼 같은데 자동 샘플 전송은 다들 끄시더라구요.

설정 -> 개인정보 -> 피드백 및 차단 – windows에서 내 피드백 요청 -> 안함처리 – Microsoft에 장치 데이터 보내기 -> 기본

시스템 복원 기능 끄기 – OS가 이상해지면 재설치하는걸 좋아하거든요!!!

최소한 이정도는 해줘야 그나마 저사향인 X80 태블릿을 사용하는데 무리가 안가네요..

더 좋은 사양의 태블릿이 가지고 싶네요!!!
