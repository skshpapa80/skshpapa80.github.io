### 1999년

* 4월 동아리 전시회 준비 프로젝트 Player GX 1.0 <br />
작업툴 : Visual Basic 5.0 <br />
간단한 멀티미디어 재생기입니다. AVI,MPG,WAV,MID,MP3 등의 파일이 재생되는 기능은 아주 단순 한 멀티미디어 재생기 처음으로 짠 프로그램..
* 9월 선배님의 프로젝트 참여 에꾸르( 재무,회계 관리 프로그램) <br />
작업툴 : Delphi 4.0 C/S <br />
DB : 오라클 7.0 <br />
프로젝트 중간부터 참여 했습니다. 프로젝트 하는법 델파이 , 오라클 만 배우다가 프로젝트에서 빠짐 참으로 아쉬웠습 니다.끝까지 같으면 좀더 많은 것들을 배울 수 있었을 텐데..

### 2000년

* 7월 “에이스 아카데미” 홈페이지 제작(아르바이트) <br />
작업툴 : 나모 3.0 , 페인트샵6.0, 플래쉬4.0 <br />
홈페이지를 배우면서 만드느라 참 힘들었습니다. 디자인도 꽝인 제가 플래쉬를 이용해서 디자인하고 하지만 지루하지 않은 방학을 보냈습니다.
* 12월 “Couple Agent” 졸업작품 제작 <br />
언어 : ASP <br />
서버 : Windows 2000 server (정보처리과 서버) <br />
작업툴 : 울트라 에디터,포토샵 5.5,페인트샵 6.0,Cute FTP <br />
DB : ACCESS 2000(MDB) <br />
한달동안 열씸히 기획 ,자료조사, 구현등 정신없이 보냈습니다. 기능은 회원관리,게시판(답변가능),카운터,투표,100일 계산 등이었습니다. 커플들을 위한 내용을 채우느라 열씸히 커플 관련 자료를 구하고.. 드디어 완성 졸업작품전에 무사히(?) 출품되었습니다.

### 2001년

* 1월 운전면허 학원 고객관리 프로그램(아르바이트) <br />
작업툴 : VisualBasic6.0 sp3, 출력툴 : 크리스탈 리포트 <br />
DB : ACCESS 2000(MDB) <br />
자동차 학원의 고객관리 프로그램, 작업기간 1달 반, 인쇄폼 5개 <br />
지하철 표지관리 시스템 개발 (첫회사) <br />
작업툴 : Delphi 5.0 <br />
미완성한 프로젝트 아파쳐라는 FM 관리툴을 이용한 도면 핸들링 프로그램..
* 9월 수자원 실시간 공사 관리 시스템 CCMS 유지보수 (첫회사) <br />
작업툴 : ASP <br />
DB : MSSQL Server 2000 <br />
제가 현제 다니고 있는 회사에서 처음 맡은 프로젝트네요. 이미 개발된 웹사이트를 유지보수를 진행했습니다.
이미지 리사이징 컴포넌트도 만들고 스크립팅 컴포넌트도 만들고 내공이 부쩍 늘었네여..
* Easy Viewer / Easy View Builder 개발 (첫회사) <br />
작업툴 : Delphi 5.0 <br />
회사에서 사용하는 파일 기반의 CD-ROM 뷰어 생성 프로그램 입니다. CD-ROM 에서 Dwg,Tiff 등의 파일들을 인덱싱후. 바로 확인할수 있는 프로그램 입니다.
