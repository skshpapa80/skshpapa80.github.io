차장님이 우리도 이젠 웹 기반의 ERP를 만들어야 한다면서 <br/>
Flex에 대해 말씀하시더군요 <br/>
여기저기 검색하다 Flex 빌더 2.0 트라이얼 버전을 받았습니다.  <br/>
간단히 설치하고 튜토리얼을 따라서 <br/>
그리드를 하나 프레임 위에다 올려놓고 XML을 연결했습니다. <br/>
UI 구성하는데 Flash를 잘 모르는 사람도 쉽게 할 수 있을 거 같았습니다. <br/> 
만들어진 파일을 실행해 보니 <br/>
XML 데이터가 그리드에 표시되고 <br/>
칼럼 정렬, 칼럼 이동 기능, 칼럼 리사이즈까지 기본 구현되어 있더군요. <br/>
(예전에 힘들게 구현해서 쓰던것들인데…) <br/>
왠지 좋아 보였습니다. 아직 깊게 보질 못해서 Flex에 대해 잘 모르지만 <br/>
앞으로 가능성이 좋아 보이더군요.   <br/>
<br/>
결론 : 그래서 나물이는 Flex를 배우게 되었습니다. ^^
