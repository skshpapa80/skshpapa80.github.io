### Tablet X80 power S 사양

Tablet X80 power S 사양은 다음과 같습니다.

* CPU : Intel Cheerytrail X5-8300
* RAM : 2GB DDR3L
* DISPLAY : 8인치 1920 * 1200 IPS Display
* CAMERA : Front/Rear Camera 2.0 MB
* OS : Windows 10 Home / Android 5.1
* STORAGE : 64G eMMC (윈도우 40G, 안드로이드 20G)
* NETWORK : 2.4Ghz WIFI, Bluetooth 4.0 Support
* BATTERY : 3800mah

고사양의 Tablet 는 아닙니다. 따지고 보면 저샤앙 Tablet이죠

장점은 듀얼 OS를 사용할수 있고 64G라 32G의 다른 Tablet에

비해 용량의 여유가 있고 해상도가 1920 * 1200이라 좋다는점

그리고 메탈유니바디로 디자인이 이쁩니다.

외장 SD카드를 128G까지 지원해서 microSD를 사용하면 용량걱정은 덜할꺼 같습니다.

micro HDMI를 지원해서 TV로 큰화면을 볼수 있습니다.

단점은 배터리가 오래가지 않고 요즘 5G 와이파이를 지원하지 않습니다.

카메라는 없다고 생각하시는게..

그리고 게임을 즐기시려는 분은 다른 Tablet를 알아보시는게 건강에 좋습니다.

### 윈도우 사용

윈도우는 업데이트만 하고 잘 사용하지 않습니다.

하지만 ActiveX를 사용해야 하는 금융권 사이트나 IE 만 지원하는 영어 강좌 사이트를 사용할때에는 주로 사용합니다.

큰딸이 <strong>영서당</strong>에서 <strong>영어 원서 읽기 공부</strong>를 하는데 안드로이드나 안드로이드 태블릿에서는 녹음을 지원하지

않는데 제 Tablet 으로는 가능합니다. 윈도우 OS로 부팅해서 IE 11을 실행하면 되니

놀러가서도 영어공부를 빼먹지 않고 할수 있을꺼 같습니다.

그리고 윈도우 스토어에서

* Microsoft Minesweeper(지뢰찾기)
* Microsoft Solitaire Collection(카드놀이)
* Microsoft Soduku (수도쿠)

이 게임들을 깔았는데 시간때우는데에 최고입니다.

### 안드로이드 사용

주 사용은 안드로이드 입니다. 가끔 윈도우 파티션을 제거하야 안드로이드 전용으로만

사용하면 어떨까 하는 생각도 가지고 있습니다.

<p align="center"><a href="/assets/images/OJERVbL.jpg" id="open-image"><img src="/assets/images/OJERVbL.jpg" alt="Tablet" width="400"/></a></p>
리디북스, XODO 앱 스크린샷

가장 많이 사용하는 앱은 리디북스와 XODO입니다.

PACKT에서 매일매일 무료로 나눠주는 PDF ebook 을 보거나

리디북스의 책을 봅니다. 스마트폰으로 보는거랑을 확실히 틀려서

책을 더 많이 보게 되더군요. 무게도 Tablet이 별로 무겁지 않아서 한손으로 들고 보는데 무리가 없습니다.

그리고 자기전에 침대에 누워서 유튜브를 보거나 

<strong>클리앙</strong>을 보는건 덤이지요..(물론 가끔은 클리앙을 가장 많이볼때도 있습니다. ^^)

아 그리고 탭진이라는 앱이 설치되어 있는데

처음에 잡지를 다운받으로 하니 다운이 안되어 사용을 못했는데

네이버에 어떤분이 올려놓으신 APK를 설치하니 정상적으로 다운로드가 되었습니다.

그래서 탭진으로 여러 잡지들도 다운받아 보고 있습니다.

거기다 다음웹툰, 퍼펙트 뷰어 등등 여러 앱들을 사용하면서 Tablet를 잘사용하고 있습니다.
